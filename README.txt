# 🌍 **Global Times - Interactive World Time**  

### **Your Gateway to Real-Time Global Clocks**  
Stay updated with the **current local time** of any country with a simple click! **Global Times** is an intuitive, interactive web application designed to make time zone tracking effortless. Whether you're scheduling meetings across time zones or just exploring world clocks, this tool provides a seamless experience.  

---

## 🚀 **Key Features**  
✔️ **Interactive World Map** – Click on any country to instantly view its **real-time local time**.  
✔️ **Live Time Updates** – The displayed time dynamically adjusts to reflect accurate global time zones.  
✔️ **User-Friendly Interface** – Smooth navigation with zoom and pan functionalities.  
✔️ **Mobile & Desktop Optimized** – Fully responsive design for seamless use on any device.  

---

## 🛠️ **Technology Stack**  
🔹 **HTML5** – Structure & layout  
🔹 **CSS3** – Modern styling & responsiveness  
🔹 **JavaScript** – Powering interactivity  
🔹 **D3.js** – Dynamic world map visualization  
🔹 **Moment.js** – Precise time zone handling  
🔹 **Leaflet.js** *(optional)* – Advanced map interactions  

---

## ⚙️ **Installation & Setup**  
Want to run this project locally? Follow these steps:  

### **1️⃣ Clone the Repository**  
```bash
git clone https://github.com/yourusername/global-times-map.git
cd global-times-map
```

### **2️⃣ Open the Project**  
Simply open **`index.html`** in your browser.  

For a local development server, use:  
📌 **Python 3 (built-in HTTP server)**  
```bash
python -m http.server
```
Now, visit `http://localhost:8000` in your browser! 🚀  

---

## 🎯 **How It Works**  
🗺️ Open the interactive world map.  
🖱️ Click on any country to instantly view its **current local time**.  
🔍 Zoom in and out for a better visual experience.  
⏳ The displayed time updates **in real-time** for accuracy.
